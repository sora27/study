<h6>4. 전역변수는 사용하지 않는다.</h6>
<h6>5. 모든 입출력은 main()에서만 한다.</h6>
<h6>6. 입력 받은 1권의 도서에 대해 ①과 ②를 호출한 후 다음 도서를 입력 받는다.</h6>
